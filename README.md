# rainiwalk.github.io

Personal website. Built with Astro + Tailwind CSS, deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # production build to dist/
npm run preview    # preview production build
```

## Adding a Project

Edit `src/data/projects.ts` and push to `main`. GitHub Actions will deploy automatically.
